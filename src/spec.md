# Specification

## Summary
**Goal:** Update all user-facing branding and default metadata to use the name “Yono Store”.

**Planned changes:**
- Replace any visible product/site name text in the public UI with “Yono Store” (e.g., header/brand label, home hero/title if present).
- Replace any visible product/site name text in the admin UI (e.g., admin header/title) with “Yono Store”, if present.
- Set the default HTML document metadata (at minimum the base document title) to “Yono Store” for routes that don’t override metadata.
- Update any in-repo user-facing documentation pages or visible onboarding/help text (if present) to reference “Yono Store”.

**User-visible outcome:** Visitors and admins consistently see “Yono Store” as the site name across pages, and the browser tab title defaults to “Yono Store”.
